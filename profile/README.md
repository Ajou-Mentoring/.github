## 서비스 소개

'ClassConnect'는 아주대학교 SW융합교육원에서 시행하고 있는 '코딩 트레이너' 사업(활동)에 참여하는 교수, 멘토, 학생의 불편함을 해소하기 위해 기획된 플랫폼 기반 서비스이다. '코딩 트레이너'의 멘토는 본 서비스를 통해 멘토링 일정 조율, 일지 작성, 증빙자료 취합 등의 작업을 간소화할 수 있으며, 교수는 수강생들의 참여도와 성취도를 확인할 수 있다.
